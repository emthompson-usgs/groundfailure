gfail.temphdf¶

class gfail.temphdf.TempHdf(grid2dfile, filename, name=None)[source]¶

Bases: object

Convert grid2d file into a temporary hdf5 file for reducing memory load.

Parameters:
  • grid2dfile – grid2d file object to save
  • filename (str) – Path to where file should be saved (recommended it be a temporary dir).
  • name (str) – Name of layer, if None, will use filename minus the extension, or if a multihazard grid2d object, each layer will have its own name.
