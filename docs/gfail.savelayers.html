gfail.savelayers¶

gfail.savelayers.savelayers(grids, filename)[source]¶

Save ground failure layers object as a MultiHazard HDF file, preserving metadata structures. Must all have the same geodict.

Parameters:
  • grids – Ground failure layers object.
  • filename (str) – Path to where you want to save this file.
gfail.savelayers.loadlayers(filename)[source]¶

Load a MultiHazard HDF file back in as a ground failure layers object in active memory (must have been saved for this purpose).

Parameters:filename (str) – Path to layers file.
